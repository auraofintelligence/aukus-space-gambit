# AUKUS Space Gambit

Public hypothesis site for testing whether the International Space Station could be preserved, reboosted, repurposed, or otherwise reviewed before the current controlled-deorbit pathway becomes irreversible.

This is not an official AUKUS, NASA, SpaceX, Australian Government or ISS partner-agency proposal.

## Pages

- `index.html` - public gateway
- `baseline.html` - current NASA transition and deorbit baseline
- `gambit.html` - AUKUS Space Gambit hypothesis
- `feasibility.html` - review gates
- `support.html` - Markdown support signal builder
- `sources.html` - source trail and public/private boundaries

## Local preview

Run a simple local server from this folder:

```powershell
python -m http.server 4190 --bind 127.0.0.1
```

Then open `http://127.0.0.1:4190/index.html`.
